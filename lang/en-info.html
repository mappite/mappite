Route Creation - How to...

create a route?
Click (or Long Tap on touchscreens) to add a via point. Right Click (or again Long Tap on touchscreens) to remove it. That's all!

The default behavior when clicking on the map is to find the closest road, to change that see below.
If you want to locate an address click on lens icon on top-right, type address/city/POI name and press enter.
add a via point among two existing points?
From a PC you can drag any pixel in the route to transform it in a new via point.
If you want the next via point to be at a specific leg (i.e. among two existing via points), from any device go to bottom-left panel, , click on + sign among two points, the sign becomes green:
move an existing via point?
From a PC you can drag any via point or to change a via point order, from any device, go to bottom-left panel, , click on red up or down arrow.
close loop or invert a route?
From bottom-left panel, click on ↺ to close a route or ⇄ to invert it. When route is in close loop mode the ↺ icon becomes green and last route point will match with first one: any new via point will be added before the last one.
see cumulative distance among via points?
Click on leg timing/distance to toggle among showing relative or cumulative figures.
add Mountain Passes, Fuel Stations, etc. to a route?
From the top-right bar click on the proper icon, available items will appear on the map. Click/Tap on icons on map to display details, Right-Click/Long-Tap to add to the route. By default it gets added at the end but if you want it among two existing via points remember to set the leg as described here To remove all icons form the map Right-Click/Long-Tap on top-right bar icon.
add points in unpaved roads/tracks?
From bottom-left panel, , select Click Anywhere. This will enable to select points outside of paved roads. The default Snap On Road tries to snap to the closest paved road.
Unpaved roads are usually slower/longer than paved so you may need to add extra points to force the route to pass on them.
When you export a route to GPX format for a GPS Navigator, make sure your navigator maps contains unpaved roads as well. For best result use OSM maps.

Sharing - How to...

share routes with fiends?
From bottom-left panel, , you can click on Create Short Link to get a short URL to share (like https://mappite.org/abcd), click on Facebook or Whatsapp icon (whatsapp this is supported in mobile devices where Whatsapp is installed), click on Forum to get the BB code to copy and paste in your Forum.
export a route to a file?
From bottom-left panel, , you can select format (gpx route/track/waypoints or itn route) and click on Export button.
Note Apple Safari users: due to a bug in some Safari versions, the file may be saved with name "Unknown", rename to "something.gpx" to use it.
transfer a mappite route to a GPS Navigator?
In general just export the route as a file, plug your navigator via USB and copy the exported file in the proper folder according to your navigator instructions (i.e. on Garmin copy the gpx file in \GARMIN\GPX folder of internal memory or SD card, unplug and turn on your navigator, go to routes menu and follow import routes link if the new route does not appear in the list).
Note: on Android smartphones you can connect your navigator to the phone with an usb otg cable and save the mappite route or track directly in the navigator... useful when travelling.
import a route or a track into mappite?
From bottom-left panel, icon, select Import and pick the file. You can import gpx tracks and gpx&itn routes. Routes can be edited, saved, exported. You can import multiple routes to collate them. Tracks are displayed on top of the map with altitude (if available) and discarded once you close the browser

Save - How to...

save a route?
From bottom-left panel, type a name and click the save button.
Routes are saved in your browser local storage, do not delete your browser local storage unless you register to mappite
