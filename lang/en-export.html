On Garmin, connect the device via USB and copy the GPX file on GARMIN/GPX/ folder, on some devices it gets imported automatically, on others go to device menu Tools->MyData->Import
On TomTom copy the ITN file in ITN/ folder (or the Route GPX in the GPX2ITN/)
Routing Tips: Make sure to set your device routing options and avoidances as defined in this mappite route. Use more via points if route differs.
